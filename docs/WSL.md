# WSL Linux

Module names:
- long: `wsl-linux`
- short: `wsl`

## Install Module

```powershell
setupx install wsl-linux
stx install wsl-linux
stx wsl
```

## List Components

```powershell
setupx components wsl-linux
```

## Components

```powershell
setupx install wsl-linux wsl
setupx install wsl-linux ubuntu
setupx install wsl-linux docker-desktop
```
