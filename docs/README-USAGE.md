# SetupX Usage

## Main Commands

```powershell
setupx help
setupx ls
setupx list-all
setupx status
setupx search docker
```

## Install Commands

```powershell
setupx install chocolatey
setupx install web-development
setupx install web-development nodejs
setupx instal pgkm chocolatey
setupx -i web-development yarn
```

## Aliases

- `stx` is the short CLI alias.
- Module aliases:
  - `web-development` / `wdev`
  - `ai-development-tools` / `aidve`
  - `common-development` / `codev`
  - `cloud-development` / `cdev`
  - `mobile-development` / `mdev`
  - `wsl-linux` / `wsl`
  - `data-science` / `dscience`

## Test Commands

```powershell
setupx test-module pgkm
setupx test-module web-development
setupx test-component pgkm chocolatey
setupx check-status
```
